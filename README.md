# Environmental Permit System

نظام إدارة تصاريح البيئة الصحية - نظام شامل لإدارة ومتابعة طلبات التصاريح البيئية للمؤسسات الحكومية والخاصة.

## المميزات

### 🏢 إدارة الملفات
- إنشاء وتعديل ملفات التصاريح البيئية
- نظام ترقيم تلقائي للملفات
- تتبع حالة الملفات (مقبول، مرفوض، تحفظ، انتظار)
- إدارة المؤسسات العامة والخاصة

### 📊 الإحصائيات والتقارير
- إحصائيات شاملة لجميع الملفات
- تقارير حسب الحالة والفترة الزمنية
- رسوم بيانية تفاعلية
- معدلات الموافقة والرفض

### 👥 إدارة المستخدمين
- نظام أدوار متعدد (مدير، موظف، مشاهد)
- تسجيل دخول آمن
- صلاحيات مختلفة حسب الدور

### 🔔 نظام الإشعارات
- إشعارات فورية للتغييرات
- تتبع النشاطات المهمة
- إشعارات مخصصة حسب المستخدم

### 📁 إدارة المستندات
- رفع وتحميل المستندات
- ربط المستندات بالملفات
- أنواع ملفات متعددة

## التقنيات المستخدمة

### Frontend
- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS** - للتصميم
- **Lucide React** - للأيقونات
- **Vite** - أداة البناء

### Backend & Database
- **Supabase** - قاعدة البيانات والمصادقة
- **PostgreSQL** - قاعدة البيانات
- **Row Level Security** - الأمان على مستوى الصفوف

### المصادقة والأمان
- **Supabase Auth** - نظام المصادقة
- **JWT Tokens** - الرموز المميزة
- **RLS Policies** - سياسات الأمان

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- npm أو yarn
- حساب Supabase

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd environmental-permit-system
```

2. **تثبيت التبعيات**
```bash
npm install
```
## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    cd server
    npm install
    cd ..
    ```

2.  **Environment Variables**:
    Create `.env` in the root directory:
    ```
    MONGODB_LOCAL_URI=mongodb://localhost:27017/nature_app
    ```
    Create `.env` in `server` directory:
    ```
    PORT=4000
    MONGODB_LOCAL_URI=mongodb://localhost:27017/nature_app
    JWT_SECRET=your_jwt_secret_key_here
    ```

3.  **Start the App**:
    Double click `start_app.bat` or run:
    ```bash
    # Terminal 1
    cd server
    npm start

    # Terminal 2
    npm run dev
    ```

## الاستخدام

### تسجيل الدخول
1. استخدم البريد الإلكتروني وكلمة المرور
2. للحصول على حساب، تواصل مع مدير النظام

### إنشاء ملف جديد
1. اضغط على "ملف جديد"
2. املأ معلومات الملف في الخطوة الأولى
3. أدخل معلومات المؤسسة في الخطوة الثانية
4. احفظ الملف

### إدارة الملفات
- عرض جميع الملفات في جدول منظم
- البحث والتصفية حسب الحالة
- تعديل وحذف الملفات
- تتبع حالة كل ملف

### عرض الإحصائيات
- إحصائيات شاملة للنظام
- توزيع الملفات حسب الحالة
- إحصائيات الفترات الزمنية
- معدلات الأداء

## الأدوار والصلاحيات

### مدير النظام (Admin)
- إدارة جميع الملفات
- إدارة المستخدمين
- عرض جميع الإحصائيات
- حذف الملفات

### موظف (Employee)
- إنشاء ملفات جديدة
- تعديل الملفات التي أنشأها
- عرض جميع الملفات
- عرض الإحصائيات

### مشاهد (Viewer)
- عرض الملفات فقط
- عرض الإحصائيات
- لا يمكن التعديل أو الحذف
